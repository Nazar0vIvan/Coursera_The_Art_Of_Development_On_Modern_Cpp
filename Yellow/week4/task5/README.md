# Перебор перестановок

### Условие
 
Дано целое положительное число `N`, не превышающее 9. Выведите все перестановки чисел от 1 до `N` в обратном лексикографическом порядке (см. пример).

### Пример

#### Ввод

```objectivec
1   3
```

### Вывод

```objectivec
1   3 2 1
2   3 1 2
3   2 3 1
4   2 1 3
5   1 3 2
6   1 2 3
```
